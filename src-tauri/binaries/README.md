# Sidecar binaries

Place the packaged binaries for your current target here.

Examples for Windows (`x86_64-pc-windows-msvc`):

- `yt-dlp-x86_64-pc-windows-msvc.exe`
- `ffmpeg-x86_64-pc-windows-msvc.exe`

Examples for Linux (`x86_64-unknown-linux-gnu`):

- `yt-dlp-x86_64-unknown-linux-gnu`
- `ffmpeg-x86_64-unknown-linux-gnu`

Examples for macOS Apple Silicon (`aarch64-apple-darwin`):

- `yt-dlp-aarch64-apple-darwin`
- `ffmpeg-aarch64-apple-darwin`
